Python Examples

Click on each section to explore different Python concepts:

  • Hello World: print('Hello, World!')
  • Basic Addition: result = 2 + 3
  • Boolean Logic: True and False

  • If Statement: if 5 > 3: print('Five is greater')
  • While Loop: i = 0; while i < 5: print(i); i += 1
  • For Loop: for item in [1, 2, 3]: print(item)
  • Nested Loop: for i in range(2): for j in range(2): print(i, j)

  • Simple Function: def greet(name): return f'Hello, {name}'
  • Lambda Function: add = lambda a, b: a + b
  • Filter with Lambda: filtered = list(filter(lambda x: x>0, [-1, 0, 1, 2]))

  • List: my_list = [1, 2, 3]
  • List Methods: my_list.append(4)
  • List Comprehension: squares = [x*x for x in range(5)]
  • Tuple: my_tuple = (1, 2, 3)
  • Set: my_set = {1, 2, 3}
  • Dictionary: student = {'name': 'Alice', 'age': 20}

  • Reading a File: with open('file.txt', 'r') as f: print(f.read())
  • Writing to a File: with open('file.txt', 'w') as f: f.write('Hello!')

  • Importing a Module: import math; print(math.sqrt(16))
  • Random Number: import random; print(random.randint(1, 10))

  • Basic Class: class Dog: pass; my_dog = Dog()
  • Class with Method: class Dog: def bark(self): print('Woof!')

  • Try-Except Block: try: print(10/0) except ZeroDivisionError: print('Cannot divide by zero')
  • Enumerate List: for index, value in enumerate(['a', 'b']): print(index, value)
  • Zip Lists: for a, b in zip([1,2], [3,4]): print(a,b)
